# 基于Python和OpenCV的给图形界面增加易用的滑动条功能
## 项目描述
基于OpenCV实现的图像处理和计算机视觉方面的通用算法，我们可以使用creatTrackbar()函数来创建一个滑动条，以调节参数，可以达到按钮的作用，是一个非常实用的功能。通过creatTrackbar()函数和回调函数的配合使用，利用滑动条功能我们可以给图形界面实现颜色、大小和是否出现等变化，并将它们进行组合，以实现不同的需求。
